The CSS display Property

display: inline

Any height and width properties have no effect on an inline element. By default, inline elements display on the same line as other inline elements. Inline elements are only as wide as they need to be, i.e., they are only as wide as their content.

display: inline-block

Like a block element, you can set the height and width of an inline-block element. Inline-block elements, though, display on the same line, just like an inline-element.

display: block

Block elements are the opposite of inline elements. Remember that inline elements display on the same line. Block elements, however, start on a new line and will take up the full width of their container. Commonly used block elements include:
  • div
  • h1 - h6
  • p
  • li
